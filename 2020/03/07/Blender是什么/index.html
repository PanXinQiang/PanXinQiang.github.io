Blender是什么

发表于 2020-03-07 | 分类于 知识 | 阅读数

Blender是什么?

  Blender是一款免费和开源的3D动画软件。
  最初，这个程式是被荷兰的一个动画工作室NeoGeo设计为内部使用的程式。但后来NeoGeo被收购，其主要程式设计者彤·罗森达尔于1998年6月成立Not a Number Technologies（NaN）公司，将其进一步发展，并以共享软体的形式对外发布这个程式，直到NaN公司于2002年宣告破产。
  在经过债权人同意后，Blender缴付一次性报酬十万欧元后变为自由软体，并以GNU通用公共许可证发布。在2002年7月18日，彤·罗森达尔开始为Blender筹集资金；同年9月7日，Blender宣布筹集足够资金，并将其源码对外公布。因此，Blender现在是自由软体，并由Blender基金会维护与更新。

  Blender支持的格式

  支持自身所用.blend之外的多种通用3D格式。

  导入

　　Collada（.dae）Motion Capture（.bvh）Scalable Vector Graphics（.svg）Stanford（.ply）Stl（.stl）Autodesk 3ds Max（.3ds）Autodesk FBX（.fbx）Wavefront（.obj）X3D Extensible 3D（.x3d）
导出
　　Collada（.dae）Stanford（.ply）Stl（.stl）Autodesk 3ds Max（.3ds）Autodesk FBX（.fbx）Wavefront（.obj）X3D Extensible 3D（.x3d）

  使用Blender的作品

  国家航空航天局 （NASA）为火星探测车“好奇”着陆三周年纪念日作出了交互web-软件。探测车的动作、摄影机和机械手的操纵是基于 Blender 和 Blend4Web 创造的软件实现的，也打了一些著名的使命事件。软件是在SIGGRAPH2015大会WebGL部分的开始在年提出的。
  第一套使用Blender专业影片是蜘蛛侠2，在该片制作中用来绘制分镜脚本。
  Blender参与制作的有很多，比如《高堡奇人》《行尸走肉》《魔兽》等。
  全由Blender制作的电影，Blender基金会的开源电影以及最近的《未来机器城》。

为什么要学习Blender

开源免费

  对于个人三维用户来说，一款正版的三维软件价格太过昂贵，使用盗版就面临着被软件开发公司发律师涵的风险，虽然一般情况下软件公司不会针对个人用户，但是如果你的作品引起了较大的关注，就可能会面临这样的风险，以Autodesk的软件Max和Maya为例，使用盗版软件制作的作品，知识产权并不属于你，购买学生版的软件，是不能够进行商业用途的，只能学习，并且Autodesk公司有权利直接打开你的电脑查看你的东西。
  对于公司来说，公司是软件公司重点关照的地方，软件的收入基本都是来自公司购买的正版，一旦你的公司稍微做大或者是引起不好，电话和律师涵可能马上就到，公司只能去花钱购买或者停止使用软件，除非你的公司装有这些软件的电脑是不链接网络的，这样可能入软件公司就查不到你了。
  但是Blender不同，GUN许可证可以说是完全为了用户，你可以免费使用Blender，软件开源；网上有传言说Blender制作的东西也需要开源，这是错误的，你使用Blender制作的作品是可以用于商业用途，产权属于用户，在这个版权越来越重要的年代，Blender的优势会越来越明显。

更新

  Blender拥有世界上最活跃的社区，Blender的开源的资金是来自于社区用户，可以说开源的Blendrr诞生于众筹，以用户为中心，开发Blender的人最初就是一个艺术家，他们不关心这个程序的代码好不好看，只要代码有效，能够达到自己的目的，那么这个代码就是可以用的。Blender的更新内容一般都是根据基金会每隔一段时间制作的开源电影，针对电影这种综合实际项目中遇到的问题来进行更新，还有就是根据社区反馈的意见。Blender可以说是最在乎用户意见的软件，因为它诞生于用户，目前的开发资金来源也是来自于公司以及个人用户的赞助。写在Blender开发计划里的东西可能会因为资金和技术的问题推迟，但一定会实现。

渲染

  Blender2.79b版本中带有几个渲染器，Blender渲染，Blender游戏，Cycles渲染以及OpenGL，一般使用的是Cycles渲染器。
  Cycles渲染器是一款无偏差的物理渲染器，支持GPU渲染，CUDA和OpenCL均可，效果与CPU渲染相同，支持PBR流程，采用的是与Maxwell，Arnold相同的路径追踪算法，无偏差渲染的优点在于效果好，全部按照物理计算，缺点在于速度较慢，但是Cycles已经支持GPU计算，所以在速度上相比其他仅支持CPU的渲染器要快许多，但是不敌Redshift这类有偏差GPU渲染器。
  Blender 2.8移除Blender游戏，添加Eevee渲染引擎，与Cycles渲染器不同，Eevee并不是通过光线追踪来进行计算的，使用的是类似游戏引擎的计算方式，基本上支持所有渲染器可以做到的效果，渲染速度极快，渲染大图可以做到以秒为单位的渲染速度，和Cycles渲染器的材质通用，缺点磨人的效果比不上光线追踪渲染器，光照的效果可能会出错，对于折射和反射需要添加类似游戏引擎的反射探针，进行烘培。

插件

  Blender自带很多插件，只需要在设置里勾选上就可以开启（记得点击保存用户设置），一些第三方插件可以通过Blendrmaker网站购买，也有很多第三方的插件开发者喜欢Blender的开源免费，所以也有很多免费的第三方插件。

教程

  可以通过斑斓中国（blendercn.org）来获取教程，教程十分的详细，基本可以让一个从来没有接触过建模软件的人学会Blender的基础，视频分布在B站，优酷，西瓜视频等平台，目前网上很多教育机构例如饼干教育，教程的价格较高，并且内容也是教基础，不推荐付费进行学习基础教程，基础可以从社区教程学习，当你需要学习进阶内容的时候可以选择进行付费学习，最近aboutcg也开始针对Blender制作教程，在我看来价格还算可以。
